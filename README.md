# READHAVEN

A books e-commerce website for all book lovers!

---

## INSTALLATION

```
1. git clone https://github.com/niharika-kesarwani/readhaven
2. cd readhaven
3. npm install
4. npm start
```

---

## FEATURES

1. Home Page
2. Product Listing Page
3. Single Product Page
4. Filters by
   - Price
   - Category
   - Ratings
5. Wishlist Management
6. Cart Management
7. Search
8. Loading & Alerts
9. Authentication
   - Sign-up
   - Login
   - Logout
10. Address Management
11. Checkout

---

## TECHNOLOGIES USED

### FRONTEND

1. React JS (useContext, useReducer)
2. React Router v6
3. Vanilla CSS
4. MUI Icons
5. React Hot Toast

### BACKEND

1. mockBee - https://mockbee.netlify.app/
